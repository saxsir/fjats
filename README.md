fjats
===
日本のアクセスランキングが高いサイトを取得するスクリプト

## About
This script fetch the top 525 sites in Japan from alexa.com.

**Please use at your own risk.**

## Environment

- Python 3.4.3
- pip
  - PyQuery

## Run

    $ make run

or

    $ python main.py
